
***"A journey of a thousand miles begins with a single step!"***

This is a very simple single-server application flow. No databases, no caching, nothing complicated going on.

![[Pasted image 20260405172442.png|627]]

The business logic is handled via server-side languages such as JS, Python, Java, etc.
The information is usually served via JSON

```JSON
GET /users/12

{
	"id": 12,
	"firstName": "John",
	"lastName": "Smith",
	"age": 55,
	"courses": ["arabic", "urdu", "english"], // arrays in json
	"address":
		{ // notice how this is nested json
			"street": 16431,
			"name": "cavendish drive",
			"city": "houston",
			"state": "texas"
		}
}
```
